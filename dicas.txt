Primeiros 3 comandos ao extrair na sua máquina:
bashcd studio-covado
cp .env.example .env.local   # edite com suas credenciais
npm install
npm run dev
O projeto não vai subir sem o banco configurado — preencha as variáveis POSTGRES_* no .env.local primeiro (você pega elas no painel da Vercel após provisionar o Postgres).
